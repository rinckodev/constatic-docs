Using the [CLI to generate your project](/docs/discord/start), you can select a server API preset.

<ImageZoom src={"/guides/cli/bot-server-api.svg"} alt="Init discord bot" width="400" height="400" />

The preset API server will be in the `server` folder inside `src`.

The available presets are:

- Fastify - [See the guide](/docs/discord/presets/servers/fastify)
- Express - [See the guide](/docs/discord/presets/servers/express)
