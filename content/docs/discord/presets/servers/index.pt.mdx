Um servidor de API (Interface de Programação de Aplicações) é um sistema que expõe dados e funcionalidades de uma aplicação por meio de endpoints acessíveis através de requisições HTTP, permitindo que diferentes sistemas, como aplicativos web, mobile ou de terceiros, se comuniquem e utilizem esses recursos de forma segura e eficiente.

Usando a [CLI para gerar seu projeto](/docs/discord/start), é possível selecionar uma predefinição de servidores de API.

<ImageZoom src={"/guides/cli/bot-server-api.svg"} alt="Init discord bot" width="400" height="400" />

O servidor de API das predefinições vai estar na pasta `server` dentro de `src`

As predefinições disponívei são:

- Fastify - [Veja o guia](/docs/discord/presets/servers/fastify)
- Express - [Veja o guia](/docs/discord/presets/servers/express)
