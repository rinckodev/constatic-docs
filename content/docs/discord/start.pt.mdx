## Requisitos

<Blockquote color="warn" icon={icons.warn}>
  É preciso seguir estes requisitos antes de começar
  - Um editor de código (de preferência o [Visual Studio Code](https://code.visualstudio.com/))
  - É necessário [NodeJs](https://nodejs.org/) 20.11 ou maior instalado (ou [Bun](https://bun.sh/))
  - Conhecimento básico de lógica de programação
  - Conhecimento básico de [javascript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) e [typescript](https://www.typescriptlang.org/docs/handbook/typescript-in-5-minutes.html)
  - Conhecimento básico de [discord.js](https://discordjs.guide/)
</Blockquote>

## Gerando o projeto

Para iniciar um novo projeto de bot de discord você pode utilizar a CLI.

<Steps>
  <Step>
    Crie uma nova pasta para o seu projeto e abra ela no seu terminal
  </Step>
  <Step>
  Use o comando abaixo para iniciar a CLI
  ```bash
  npx constatic@latest
  ```
  </Step>
  <Step>
    Selecione `Iniciar projeto de bot de discord`
    <ImageZoom src={"/guides/cli/bot-init.svg"} alt="Init discord bot" width="400" height="400" />
  </Step>
  <Step>
    Digite o nome do projeto
    <Blockquote title="Dica" color="tip" icon={icons.bulb}>
      Escreva `.` para criar o projeto no diretório atual e usar o nome dele
    </Blockquote>
    <ImageZoom src={"/guides/cli/bot-name.svg"} alt="Bot name" width="400" height="400" />
  </Step>
  <Step>
    Selecione uma predefinição de banco de dados (opcional)
    <ImageZoom src={"/guides/cli/bot-db.svg"} alt="Bot db" width="400" height="400" />
  </Step>
  <Step>
    Selecione recursos extras (opcional)
    <ImageZoom src={"/guides/cli/bot-features.svg"} alt="Bot features" width="400" height="400" />
  </Step>
  <Step>
    Selecione o seu gerenciador de pacote para instalar as dependências (opcional)
    <ImageZoom src={"/guides/cli/bot-deps.svg"} alt="Bot deps" width="400" height="400" />
  </Step>
  <Step>
    E pronto! O seu projeto foi gerado e você já pode começar a desenvolver o seu bot.
  </Step>
</Steps>

## Configurando o projeto

Com o projeto gerado, coloque o token do seu bot no arquivo `.env` que está na raiz do projeto
<Blockquote color="info">
    [Veja como criar uma aplicação de bot e obter o token no portal de desenvolvedor do discord](/discord/guides/application)
</Blockquote>

Depois de colocar o token você pode executar o projeto em desenvolvido utilizando o comando abaixo

```
npm run dev
```

Após isso o seu bot deverá ficar online e você já pode começar o desenvolvimento!
