## Requisitos

<Callout type="warn">
  É preciso seguir estes requisitos antes de começar
  - Um editor de código (de preferência o [Visual Studio Code](https://code.visualstudio.com/))
  - É necessário [NodeJs](https://nodejs.org/) 20.11 ou maior instalado (ou [Bun](https://bun.sh/))
  - Conhecimento básico de lógica de programação
  - Conhecimento básico de [javascript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) e [typescript](https://www.typescriptlang.org/docs/handbook/typescript-in-5-minutes.html)
  - Conhecimento básico de [discord.js](https://discordjs.guide/)
</Callout>

## Gerando o projeto

Para iniciar um novo projeto de bot de discord você pode utilizar a CLI.

<Steps>
  <Step>
    Crie uma nova pasta para o seu projeto e abra ela no seu terminal
  </Step>
  <Step>
  Use o comando abaixo para iniciar a CLI
  <CliCommand packageName="constatic@latest" />
  </Step>
  <Step>
    Selecione a primeira opção: `Iniciar projeto de bot de discord`
  </Step>
  <Step>
  Serão feitas algumas perguntas para preparar o seu projeto:
  ```md
  - Nome do projeto (caminho para a pasta)
  - Banco de dados (opcional)
  - Recursos extras (opcional)
  - Instalar dependências (opcional)
  ```
  </Step>
  <Step>
    Depois que responder tudo, seu projeto será gerado e você já pode começar a desenvolver.
  </Step>
</Steps>

## Configurando o projeto

Com o projeto gerado, coloque o token do seu bot no arquivo `.env` que está na raiz do projeto
<Callout type="info">
    [Veja como criar uma aplicação de bot e obter o token no portal de desenvolvedor do discord](/docs/discord/guides/application)
</Callout>

```properties title=".env"
BOT_TOKEN=seutoken
```

Depois de colocar o token você pode executar o projeto em desenvolvido utilizando o comando abaixo

```bash
npm run dev
```

Após isso o seu bot deverá ficar online e você já pode começar o desenvolvimento!
