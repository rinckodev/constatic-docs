# Criando um banco de dados Firestore

Você precisa ter uma conta Firebase!

Se você não tiver uma, acesse https://console.firebase.google.com/ e faça login com sua conta Google. 

Vamos criar um projeto.

<ImageZoom src={"/firestore/step-01.png"} width={500} height={500} alt="firestore-setp-01" />

De um nome ao seu projeto e continue.

<ImageZoom src={"/firestore/step-02.png"} width={500} height={500} alt="firestore-setp-02" />

Deixe essa opção desmarcada se você preferir.

<ImageZoom src={"/firestore/step-03.png"} width={500} height={500} alt="firestore-setp-03" />

Agora clique em `Firestore Database` na barra lateral.

<ImageZoom src={"/firestore/step-04.png"} width={500} height={500} alt="firestore-setp-04" />

Crie um novo banco de dados.

<ImageZoom src={"/firestore/step-05.png"} width={500} height={500} alt="firestore-setp-05" />

Deixe `production mode` marcado e clique em próximo.

<ImageZoom src={"/firestore/step-06.png"} width={500} height={500} alt="firestore-setp-06" />

Escolha o local do seu banco de dados (é recomendado uma localização mais proxima da sua aplicação).

<ImageZoom src={"/firestore/step-07.png"} width={500} height={500} alt="firestore-setp-07" />

Agora você tem um banco de dados Firestore.

<ImageZoom src={"/firestore/step-08.png"} width={500} height={500} alt="firestore-setp-08" />

Acesse as configurações do projeto.

<ImageZoom src={"/firestore/step-09.png"} width={500} height={500} alt="firestore-setp-09" />

Vá para contas de serviço e clique em gerar nova chave privada.

<ImageZoom src={"/firestore/step-10.png"} width={500} height={500} alt="firestore-setp-10" />

Você tem que salvar este arquivo no seu computador.

<ImageZoom src={"/firestore/step-11.png"} width={500} height={500} alt="firestore-setp-11" />

Renomeie o arquivo para `firebase.json` e salve.

<ImageZoom src={"/firestore/step-12.png"} width={500} height={500} alt="firestore-setp-13" />

Coloque esse arquivo na raiz do seu projeto.

Está pronto para uso.
