[Discloud](https://discloud.com/) é um serviço de hospedagem de aplicações em nuvem.

<Accordions>
    <Accordion title="Certifique-se de que o projeto está buildado" id="build">
    Para poder enviar, antes o seu projeto precisa ser buildado usando o comando abaixo:
    ```bash
    npm run build
    ```
    </Accordion>
</Accordions>

Ao gerar o projeto usando a [CLI](/docs/discord/start), é possível selecionar o recurso `discloud project`, que fará com que os dois arquivos necessários para enviar sua aplicação para a hospedagem, sejam incluídos no projeto.

<Accordions>
    <Accordion title="Mas... Se você não marcou essa opção, veja como os arquivos devem ficar">
    Crie o arquivo `discloud.config` na raiz do projeto.
    ```yml title="discloud.config"
    NAME=Awesome Bot
    TYPE=bot
    MAIN=build/index.js
    RAM=100
    VERSION=latest
    AUTORESTART=false
    START=npm run start
    ```
    Crie o arquivo `.discloudignore` na raiz do projeto.
    ```yml title=".discloudignore"
    # Folders
    /.vscode
    /src
    /deprecated
    /discloud
    /node_modules

    # Files
    /.env.dev
    /.env.example
    /.eslintrc.json
    /.gitignore
    /tsconfig.json
    /package-lock.json
    /README.md

    # Extensions
    *.md
    *.zip
    *.exe
    *.dev.json
    ```
    </Accordion>
</Accordions>

No arquivo `discloud.config` você pode definir um nome e um avatar para a sua aplicação na hospedagem se preferir.
```yml title="discloud.config"
    NAME=My Awesome Bot
    AVATAR=https://i.imgur.com/L7Mxhoj.gif
    # ...
```

Instale a extensão `discloud` no Visual Studio Code

<ImageZoom src={"/guides/discloud/extension.png"} width={380} height={200} alt="step" />

Clique na aba `discloud` na lateral do Visual Studio Code e coloque <Link href={settings.links.discloud.get_token}>seu token de api da discloud</Link>.
Depois basta clicar em `Upload Discloud` na barra inferior do Visual Studio Code.
<ImageZoom src={"/guides/discloud/upload-button.png"} width={380} height={200} alt="step" />

Se seu projeto não conter nenhum erro, ele será iniciado em alguns instantes

<Blockquote icon={icons.info} color="warn">
    Caso não queira usar a extensão do Visual Studio Code, existem outras [maneiras de hospedar e você pode conferir aqui](https://docs.discloud.com/suporte/hospedar). Apenas lembre-se de seguir as exigências abaixo: 
    - A pasta que deve ser enviada é a `build`, não `src` ([Por isso é preciso buildar antes](/discord/hosting/discloud#build)). 
    - Os arquivos e pastas que devem ser enviados são `build`, `.env`, `package.json` e `discloud.config`
    - Não envie a pasta `node_modules`.
    - É necessário definir que o script para iniciar seja `npm run start`.
</Blockquote>
