Os comandos são formas nativas de interagir com bots de Discord. Existem 3 tipos de comandos acessíveis em diferentes interfaces: **A entrada do chat**, o **menu de contexto de uma mensagem** (clicar com o botão direito em uma mensagem) e o **menu de contexto de um usuário** (clicar com o botão direito em um usuário).

Comandos de entrada do chat tem as opções de autocomplete que podem ser respondidas de forma simples usando essa base de bots.
