# Lista de emojis

É possível listar todos os emojis da aplicação com a opção `Listar`. Você verá no terminal uma lista com o tipo do emoji (se é animado ou estático), nome e id. Assim você pode conferir os emojis existentes antes de fazer qualquer outra operação como `Enviar`, `Atualizar` ou `Excluir` emojis.
