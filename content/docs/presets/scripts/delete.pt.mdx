# Excluindo predefinições de scripts

Você pode excluir predefinições caso não queira mais usando essa opção

Depois de [acessar o menu de predefinições de scripts](/docs/presets/scripts/start) você pode selecionar a opção para excluir:

<CliMenus.Scripts lang="pt" selected="delete"/>

Você poderá selecionar todas as predefinições que quiser excluir. Após selecionar e confirmar, todos os arquivos armazenados das predefinições armazenados na CLI serão excluídos.
